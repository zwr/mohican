What is Mohican?

Mohican is a UI Framework that uses Angular and Bootstrap to make your back end application fly on the front end. It's Angular directives and services make it very simple to present back end documents on front end. Eager loading will load the the documents before you need them, and its grid will let you filter, sort and select them with a speed of thought.

Mohican menu Still under development!

  • Mohican menu is a specialized Boostrab navbar that services Mohican purpose. It will work flawlessly on any display size by collapsing and reappearing as space allows.
  • You can add your own menu items and group them to any depth.
  • If space allows, items are shown as a standard menu, once collapsed they are shown as accordion.

Mohican grid

  • Mohican grid is a table like presentation of a collaction of documents
  • It took only 15 HTML tags to craete the Orders grid page in our example, yet it includes:
    • Pager for page navigation
    • Sorting by clicking the header
    • Filtering loaded documents by field or fields
    • Context sensitive popup menu
    • Dialogs for quick operations
    • In-grid document editing
    • User-specific and back-end controlled column layout
    • User-specific and back-end controlled filters

Mohican forms

Mohican forms are for detailed document viewing and editing.

  • Bootstrap forms allow pleasent UX design and state management.
  • Transaction based document statemanagement and full frontend rollback.
  • Tabbed design for documents with many details.
  • Subdocument grid views, in tabs or not.

Up and coming! New features!

  • Document creation, editing, destruction.
  • Notifications
  • Document pre-loading filtering
  • User management
  • CSS-free development using bootstrap only! Needs a bit of a budget!
